# Neon Cyberpunk Portfolio Template

## Overview
A futuristic, high-tech portfolio template featuring neon glow effects, cyber grid backgrounds, glitch animations, and terminal-style elements. Perfect for developers, tech enthusiasts, and creative professionals who want to showcase their work with a bold, cutting-edge aesthetic.

## Features

### Visual Effects
- **Neon Glow Text** - Cyan and pink neon text effects with pulsing animations
- **Cyber Grid Background** - Animated grid pattern background
- **Glitch Animation** - Text glitch effect on hover
- **Holographic Cards** - Semi-transparent cards with gradient borders
- **Scanline Effect** - Retro CRT scanline overlay
- **Floating Animations** - Smooth floating motion for profile photo

### Typography
- **Orbitron** - Futuristic display font for headings
- **Rajdhani** - Tech-style font for body text
- **Share Tech Mono** - Monospace font for terminal-style elements

### Colors
- **Neon Cyan** (#00f0ff) - Primary accent color
- **Neon Pink** (#ff00ff) - Secondary accent color
- **Neon Green** (#00ff41) - Terminal text color
- **Cyber Dark** (#0a0a0f) - Background color

### Sections
1. **Hero** - Full-screen intro with profile photo, name, and tagline
2. **About** - Profile info and stats in terminal style
3. **Education** - Academic background with cyber styling
4. **Experience** - Work history with terminal text formatting
5. **Projects** - Grid of projects with hover effects
6. **Skills** - Progress bars with neon gradients
7. **Certifications** - Achievement cards
8. **Contact** - Neon-styled contact information

### Responsive Design
- Fully responsive from mobile to desktop
- Touch-friendly navigation
- Optimized neon effects for smaller screens

## Technologies Used
- Tailwind CSS
- GSAP (GreenSock Animation Platform)
- Custom CSS Animations
- Jinja2 Templating

The template supports customization through the theme variables:
