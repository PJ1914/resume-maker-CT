# Clean Professional Portfolio Template

A modern, responsive portfolio template with smooth animations and professional design.

## Features

- ✅ **Fully Responsive** - Works perfectly on all devices
- ✅ **Smooth Animations** - Intersection Observer scroll animations
- ✅ **Fixed Navigation** - Auto-hiding navbar on scroll
- ✅ **Modern Design** - Clean and professional UI
- ✅ **Interactive Elements** - Hover effects and transitions
- ✅ **SEO Optimized** - Semantic HTML structure
- ✅ **Print Ready** - Optimized for PDF export

## Technologies

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Google Fonts (Poppins + Source Sans Pro)

## Sections Included

1. **Navigation** - Fixed navbar with smooth scroll links
2. **Hero** - Eye-catching gradient header with CTA buttons
3. **About** - Personal introduction with stats
4. **Experience** - Timeline layout for work history
5. **Projects** - Grid layout with project cards
6. **Skills** - Categorized technical and soft skills
7. **Education** - Academic background cards
8. **Certifications** - Professional certifications
9. **Contact** - Contact information with icons
10. **Footer** - Copyright and credits
